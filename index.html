Adaptive Contact-Rich Manipulation Under Changing Physicssolo project, full repo + paper + site
2026
A stability-constrained neural adaptive controller tested for recovery when contact physics drift mid-task, run across four MuJoCo environments including the real OGBench cube-single benchmark with the UR5e arm in the loop. A bounded radial-basis network sits inside a Lyapunov-derived update law, so the network supplies the features and the law supplies the stability. On the gravity-loaded arm it returns tracking error to 4.4 degrees where the fixed baseline stays at 27.2 and an unconstrained ablation diverges, and on OGBench it halves joint tracking error and lands the cube more than twice as close to the goal while the unconstrained version at the same gain goes fully unstable. The contribution is the demonstration that the constraint lets you run the adaptation gain high enough to recover fast without the instability the same gain produces without it. Five seeds, every number traced to saved data, honest about where the method ties or loses. paper + code + site

Differentiable physics-informed power-routing compilerindependent, early build
2026 – Present
An in-progress project to solve AC power-flow routing faster than legacy industrial suites by writing the real and reactive power-balance and voltage-limit equations in JAX for automatic differentiation, then training a network that maps fluctuating load demand to generation setpoints with the differentiable physics engine as the loss layer so the outputs respect physical constraints by construction. Built on standard benchmark grids such as the IEEE 14-bus and 118-bus test cases. Early stage, framed honestly as direction rather than result.
